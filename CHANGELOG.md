# Changelog

## 0.1.0

- Initial TokenScope MVP scaffold.
- Added dashboard, models, sources, timeline, sessions, and settings views.
- Added deterministic demo data, frontend aggregation, formatting, and range utilities.
- Added SQLite schema, Tauri commands, CSV import, Codex parser, and Claude Code parser.
- Added frontend tests, Rust parser fixtures, CI, release workflow, and project documentation.
- Added unsigned draft GitHub Release workflow for macOS and Windows installers.
- Added privacy regression fixtures to prevent prompt/response content persistence.
- Demo data is disabled by default and labeled as demo usage instead of real connector usage.
